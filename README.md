# 🏠 PG Life

A Full Stack PG Accommodation Booking Website built using **PHP, MySQL, Bootstrap, JavaScript and AJAX**.

---

## 📌 Project Overview

PG Life is a web application that helps users search for Paying Guest (PG) accommodations in different cities, view property details, mark properties as interested, update their profile, and request bookings online.

---

## 🚀 Features

- 🔐 User Signup & Login
- 👤 User Dashboard
- ✏️ Edit Profile
- ❤️ Interested Properties
- 🏠 Property Listing
- 🔍 Property Details
- ⭐ Property Ratings
- 💰 Rent Sorting
- 📍 City-wise Property Search
- 📅 Booking Request System
- ⚡ AJAX-based Interested Property Toggle
- 💾 MySQL Database Integration
- 📱 Responsive Design using Bootstrap

---

## ▶️ How to Run

1. Install XAMPP.
2. Copy the project into:xampp/htdocs/


3. Import:
           database/pg_life.sql
            into phpMyAdmin.

4. Start Apache and MySQL.

5. Open:


http://localhost/PGLife/
